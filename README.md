# Cosmetics Website

A modern e-commerce website for cosmetics built with React and Supabase.

## Features
- Product catalog with categories
- Shopping cart functionality
- User authentication
- Admin dashboard
- Order management
- Responsive design

## Technologies
- Frontend: React, TypeScript, Vite
- UI: Tailwind CSS, Shadcn UI
- Backend: Supabase (PostgreSQL)
- State Management: React Context
